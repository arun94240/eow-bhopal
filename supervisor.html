Save kaam nahi kar rahaRequired fields (*) sab bhare hain check, refresh, retry

🔐 Security Golden Rules:
  • Apna password KISI ke saath share NA karen
  • Public/shared computer pe logout zaroor karen
  • 5 wrong attempts → 10 min lock automatic
  • Password leak ka shaq → turant SP saab ko bataen
  • Login URL public WhatsApp groups me NA share karen
📱 Mobile Use ke liye:
  • Sirf apne personal phone par use karen
  • Public WiFi se sensitive data avoid karen
  • Screenshot share karte time PII (names, contact) hide karen
  • "Save Password" browser feature shared device pe USE NA karen
✅ Logout Karna Bhool Gaye:
Top right me apna naam ke saath "Logout" button hai. Office se nikalte time, ya dusra koi computer use karne se pehle — zaroor logout karen.
